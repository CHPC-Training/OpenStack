# Deploying Applications

## 3.1. Monitoring with Grafana and Prometheus

- Login to the VM that you want the Grafana and Prometheus services to be hosted on.
- Add the relevant repository to the package manager
  ```bash
  $ sudo dnf config-manager --add-repo https://download.docker.com/linux/centos/docker-ce.repo
  ```
- Install the relevant packages
  ```bash
  $ sudo dnf install docker-ce docker-ce-cli containerd.io docker-compose-plugin
  ```
- Start the docker service
  ```bash
  $ sudo systemctl start docker 
  ```
- Enable the service to run whenever the system reboots
  ```bash
  $ sudo systemctl enable docker 
  ```
- Check that the docker command is available
  ```bash
  $ sudo docker ps
  ```
- Create a directory to store relevant files
  ```bash
  $ sudo mkdir /opt/monitoring_stack 
  ```
